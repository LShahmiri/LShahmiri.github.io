##  Project Overview
This project presents an interactive **Earthquake Analytics Dashboard** built using **Tableau** to explore global earthquake data.  
The dashboard visualizes earthquake distribution, magnitude patterns, and geographical trends across different regions.

##  Key Features
- Global earthquake map visualization
- Top 10 largest earthquakes table
- Regional earthquake distribution
- Frequency, average magnitude, and maximum magnitude analysis
- Interactive filtering by location and date

##  Tools Used
- Tableau Public
- Data Visualization
- Geospatial Analysis
